Fantasy Football Projector
I started a project to create a fantasy football predictor using Python. By leveraging machine learning and historical player data, I aimed to forecast player performance based on stats, injuries, and matchups. Using Pandas for data manipulation and Scikit-learn for building models, I trained the algorithms on years of data to spot trends. The end result is a user-friendly app that offers weekly projections, helping fantasy managers make smarter lineup decisions. With visualizations from Matplotlib, users can easily understand the data, and the predictor updates continuously throughout the season to stay relevant.
